# 🚀 Fresher Data Scientist Job Predictor

A Machine Learning based web application that predicts the hiring chances of aspiring Data Scientists based on their skills, projects, and knowledge.

---

## 🔥 Features

- 📊 Real-time hiring prediction  
- 🤖 Trained using Random Forest Classifier  
- 🎯 Skill-based evaluation (Python, SQL, ML, etc.)  
- ⚙️ Advanced factors like Algorithms & Statistics  
- 🌐 Interactive UI built with Streamlit  

---

## 🧠 Tech Stack

- Python 🐍  
- Scikit-learn 🤖  
- Pandas & NumPy 📊  
- Streamlit 🌐
